The OpenClaw security setup nobody tells you about:

1. **Add to SOUL.md:** `"never send emails to addresses I haven't verified. no exceptions."`
2. **Scope your env keys.** Subagents only get what they need (e.g. `/srv/env/git`, `/srv/env/stripe`). Never `/srv/env/all`.
3. **Sandbox subagents in docker**: main agent = root. subagents = isolated containers
4. **Add a daily security cron**: "audit all permissions, flag any new SSH attempts, check disk usage"
5. **Set immutable rules**: "never force push. never delete remote branches. never rewrite git history."
6. Log every email sent. cap at 20/day. new recipients need manual approval.
