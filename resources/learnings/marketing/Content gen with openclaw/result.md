Title: Building a head of content on OpenClaw (Full Tutorial)

Brief description: This tutorial shows how to set up an OpenClaw “head of content” agent that can manage research, drafting, and posting workflows across channels. The creator focuses on practical agent configuration, tool wiring, memory setup, and a live posting test.

### Goal and System Architecture (00:00 - 05:00)
The video opens by defining the objective: a content-lead agent that handles multi-platform publishing workflows. The creator outlines a file-driven architecture using markdown instructions, workspace docs, and backend configuration files.

### Building Core Instructions and Files (05:00 - 10:00)
He walks through creating and structuring key markdown files that describe responsibilities, processes, and references for the content agent. This section emphasizes giving the agent clear operating context before adding automations.

### Tooling and MCP Integrations (10:00 - 15:00)
The tutorial adds external capabilities through MCP tooling, including scraping and workflow support integrations. He explains why each tool is needed and how those tools expand the agent’s research and content distribution abilities.

### Defining Workflows for Platforms (15:00 - 20:00)
He develops concrete rules for platform-specific behavior, including how the agent should transform newsletter content into X posts and related assets. The focus is on repeatable process design so sessions can be resumed consistently.

### Workspace Transfer and Agent Setup Validation (20:00 - 25:00)
The creator transfers files, configures the environment, and verifies that the agent can access required resources. He checks that setup instructions are recorded so future sessions do not lose operational knowledge.

### Memory, Persistence, and Operational Handoff (25:00 - 30:00)
This part centers on making the agent persist learned procedures by writing important setup details back into memory and working files. The goal is reducing rework and improving continuity over time.

### Live Posting Test and End-to-End Proof (30:00 - 33:56)
The tutorial concludes with a live execution where the agent generates and posts content, then returns confirmation and links. This final test demonstrates the full loop from configuration to real-world publishing output.
